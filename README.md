# admin-odin
# Made for The Odin Project

Link to page: https://ethan-hannum.github.io/admin-odin/

Page was designed on a 1080p monitor, higher or lower resolutions may break the page layout.
Didn't change much of the design layout from the given image because I knew just getting this design would be a
challenge, so I tried to add some extra responsiveness to the page.

Tough project to get right with the layout and making everything look nice, but very happy I got it done.
Using grid and flexbox together really showed me how well they work with each other. Getting the transitions right will
really help me in the future to make more responsive pages.
